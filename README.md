# SARA – Situational Awareness & Response Agent

AI-powered harassment reporting & real-time situational safety platform for women.

---

## 🚨 The Problem

- 1 in 3 women face workplace harassment
- 80% incidents go unreported
- Existing safety apps focus only on location — not context

Safety is not just *where you are*.  
It’s *what is happening around you right now.*

---

## 💡 The Solution

SARA combines:

- 🧠 Situational AI Risk Engine  
- 👭 Verified NightOwl Buddy Network  
- 🎙 Silent Distress Detection  
- 🚨 AI-Powered Multi-Action SOS  
- 📊 Anonymous Pattern Intelligence Dashboard  

SARA asks:  
**“What is happening to you right now?”**

---

## 🔥 Core Features

### 1️⃣ Situational Risk Engine
Real-time safety score using:
- Time
- Weather
- Crowd density
- Live incident reports

### 2️⃣ NightOwl Buddy Network
- Verified women matched on same route
- Live shared tracking
- Auto-alert if movement stops

### 3️⃣ Silent Sentinel Mode
Voice-triggered SOS without touching the phone.

### 4️⃣ AI SOS Engine
Within 8 seconds:
- Records audio
- Streams live location
- Alerts emergency contacts
- Notifies safe havens

### 5️⃣ Anonymous Pattern Intelligence
- Zero-login reporting
- ML clusters incident patterns
- Institutions see heatmaps, not identities
